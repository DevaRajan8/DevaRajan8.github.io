Design and Implementation of an Autonomous Self-Controlled Delivery Robot with an OTP-Based Secure Delivery

Authors: Devarajan S

This paper presents the design and implementation of a self-controlled delivery robot designed for autonomous navigation and secure last-mile delivery. The system utilizes GPS-based positioning along with an A* path planning algorithm to determine an optimal path to the target location. Computer vision methods alongside a forward-facing HD camera provide lane keeping and real-time obstacle detection by employing a YOLOv8 model. Hazards on the road initiate a local bypass procedure to realign its trajectory dynamically. During delivery, the system requests a one-time password (OTP) from the receiver, which will activate an onboard lock mechanism for the release of the parcel.
